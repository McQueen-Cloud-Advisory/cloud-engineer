# Azure

This section applies cloud engineering fundamentals using Azure.

The provider path is organized into three areas:

- [Services](services/index.md)
- [Projects](projects/index.md)
- [Patterns](patterns/index.md)

## How to Use This Section

Start with the getting started guide, review the roadmap, then work through services as they appear in projects.

## Current Focus

The first-pass content focuses on:

- Identity and access
- Object storage
- Serverless compute
- Serverless containers
- API entry points
- NoSQL or application data storage
- Secrets management
- Monitoring and observability
- Scheduled workloads
- Analytics platforms
- AI and agentic workloads
