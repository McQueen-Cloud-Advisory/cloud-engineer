# AI and Agents

## Purpose

This page introduces the core ideas behind modern AI and agentic workloads from a cloud engineering perspective.

## What You Should Understand

- Foundation models, prompts, context management, retrieval, and tools are separate moving parts that need to be designed together.
- Guardrails, evaluation, observability, and security are required for production AI workloads.
- Agentic systems increase integration and operational complexity, not only model capability.
## How This Fits Into Cloud Engineering

Modern cloud engineers need to understand how AI and agentic workloads are deployed, secured, monitored, governed, and integrated into real systems. This page frames those workloads as part of cloud engineering, not a separate discipline.
